Repetição while

Quantas maçãs não caíram na cabeça de Newton antes de ele ter percebido a pista! – Robert Frost

Se determinado código precisa ser executado várias vezes, você pode usar estrutura de controle chamada de repetição, também conhecida como laço ou loop. Cada vez que um código é executado em uma estrutura de repetição, ocorre uma iteração, ou ciclo. Uma propriedade da repetição na programação, é que em um momento ela deve parar. Se não parar, acontece o que chamamos de loop infinito. (cuidado para não ficar preso no loop infinito!).

Bohm e Jacopini demonstraram que todos os programas poderiam ser escritos com apenas três tipos de estruturas de controle (ou instruções de controle) — a estrutura de sequência, a estrutura de seleção e a estrutura de repetição.

Dentre as estruturas de repetição do Javascript, as mais usadas são while e for. Vamos começar com o while.

// instruções que vão
// antes do while
// são executadas
// normalmente

while (condicao) {
    // corpo da repeticao
    // será executado
    // enquanto a condição
    // é verdadeira
}
// instrução que vão
// depois do while
// são executadas
// normalmente

Observe sua semelhança com o if. O corpo do if executa apenas uma vez, já o corpo do while pode executar muitas vezes.

Enquanto a condição do while é verdadeira (isto é, qualquer valor diferente de false), o código do corpo do while será executado (isto é, o que estiver entre as chaves do while).

Assim como qualquer estrutura que tem um corpo, quando houver apenas uma instrução, não é necessário colocar chaves. Mesmo assim, é recomendável abrir um recuo na linha de baixo para explicitar o corpo do while.

Certifique-se de que o corpo do while altere algo na condição para que, eventualmente, a condição se torne falsa e o programa continue com as instruções que seguem o while (não caia no loop infinito!).

Assim como o if, o while pode nunca ser executado! Se a condição for falsa pela a primeira vez, o corpo do while não será executado.

Repetição controlada por contador

A repetição controlada por contador utiliza uma variável chamada contador para controlar o número de vezes que as instruções são executadas.

A repetição controlada por contador costuma ser chamada de repetição definida, porque o número de repetições é conhecido antes de o loop começar a executar.

Considere o seguinte problema: Dadas as notas (reais entre 0 e 10) de cinco estudantes de uma classe, qual a média da classe?

import scanf from "scanf"

// Inicialize as variáveis
let total = 0
let contador = 1

// Use um loop para ler as notas dos cinco estudantes
while (contador <= 5) {
    console.log(`Digite a nota do ${contador}º estudante (entre 0 e 10): `)
    let nota = scanf("%f")
    total = total + nota
    contador = contador + 1
}

// Calcule a média
let media = total / 5

// Escreva a média da classe
console.log("A média da classe é:", media)

O total é uma variável acumuladora: utilizada para acumular a soma de vários valores. Variáveis utilizadas para armazenar totais normalmente são inicializadas com zero antes de serem utilizadas em um programa.

Antes de atribuir um valor de uma variável x a uma variável y, você deve inicializar a variável x, isto é atribuir um valor a x. Por exemplo, na linha 11, a variável total recebe o valor dela mais o valor de nota, isso só é possível porque na linha 4 tínhamos atribuído um valor inicial à variável total.

Repetição controlada por sentinela

Considere o seguinte problema: Dadas as notas (reais entre 0 e 10) de um número arbitrário de estudantes de uma classe, qual a média da classe?

Como podemos determinar quando parar de ler as notas? Como saber quando calcular e imprimir a média da classe? Uma maneira de resolver esse problema é utilizar uma variável especial chamado sentinela (também chamada de variável de sinal, variável fictícia ou flag) para indicar “final de entrada de dados”. Deve-se escolher uma sentinela que não possa ser confundida com um valor aceitável de entrada.

let total = 0
let contador = 0
console.log("Digite a nota do estudante (ou -1 para encerrar): ")
let nota = scanf("%f")
 
while (nota != -1) {
    if (0 <= nota && nota <= 10) {
        total = total + nota
        contador = contador + 1
    } else {
        console.log("A nota deve estar entre 0 e 10. Tente novamente.")
    }
    console.log("Digite a nota do estudante (ou -1 para encerrar): ")
    nota = scanf("%f")
}
 
if (contador > 0) {
    let media = total / contador
    console.log("A média da classe é:", media)
} else {
    console.log("Nenhuma nota válida foi inserida.")
}

Atribuição composta

É muito comum em programação atualizar o valor de uma variável. Por exemplo, você ainda vai usar bastante essa expressão:

i = i + 1

A expressão anterior não significa que i é igual a i+1. Significa que o valor de i é alterado para seu valor atual mais 1.

Suponha que queiramos adicionar 25% ao valor de uma variável. Podemos fazer:

salario = salario * 1.25

Como essas operações são comuns no dia-a-dia de um programador, o Javascript fornece os operadores de atribuição composta (compound assignment):

Operador Exemplo Equivalente
+= x += y x = x + y
-= x -= y x = x - y
*= x *= y x = x * y
/= x /= y x = x / y
%= x %= y x = x % y
**= x **= y x = x ** y

Os operadores de atribuição composta possuem a mesma prioridade que o operador de atribuição simples (=), que possui a menor prioridade entre os operadores.

Operadores de incremento/decremento

Incrementar ou decrementar uma variável significa aumentar ou diminuir o seu valor em uma unidade.

Há dois tipos de incremento/decremento: prefixado e pós-fixado. Tanto o operador de incremento/decremento prefixado quanto o pós-fixado incrementa/decrementa o valor da variável em uma unidade.

No incremento/decremento prefixado, o resultado da operação será o valor da variável "depois" que ela for incrementada.

No incremento/decremento pós-fixado, o resultado da operação será o valor da variável antes de ser incrementada.

Exemplo Operação
++var Incremento prefixado
--var Decremento prefixado
var++ Incremento posfixado
var-- Decremento posfixado

let x = 1;
let y = ++x; //x=x+1; y=x;
let z = x++; //z=x; x=x+1;

console.log(x + " " + y + " " + z);

x++; //x=x+1;
++x; //x=x+1;
y = x--; //y=x; x=x-1;
z = --x; //x=x-1; z=x;

console.log(x + " " + y + " " + z);

    O programa deve:

    • Ler o nome e o resultado de cada estudante, sendo o resultado um valor que indica se o estudante foi aprovado ou reprovado no exame.
    • Escrever a quantidade de estudantes aprovados, a quantidade de reprovados, e se mais de oito estudantes foram aprovados, escrever “Bônus ao instrutor!”.
