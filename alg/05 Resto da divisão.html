Resto da divisão

O resto é consequência.

O operador % realiza a operação de resto da divisão, que algumas pessoas também chamam de módulo. Em várias linguagens há um operador específico para divisão inteira, isto é, aquela em que não há parte decimal, mas esse não é o caso do Javascript. Para realizar a divisão inteira em Javascript, é necessário usar o comando Math.trunc:

let dividendo = 17;
let divisor = 5;
let cociente = Math.trunc(dividendo / divisor);
let resto = dividendo % divisor;
console.log(dividendo, ' = ', divisor, ' * ', cociente, ' + ', resto);
// 17 = 5 * 3 + 2

Observações:

  • Atente-se para a fórmula, numa divisão inteira, dividendo = divisor * cociente + resto.
  • O resto sempre será menor que o divisor.
  • O resto de um número dividido por dez é seu último dígito.
  • O cociente de um número dividido por dez são todos os seus dígitos menos o último.
  • O resto da divisão de um número menor por um número maior é sempre o número menor.
  • A divisão inteira (Math.trunc) sempre gera números inteiros, mesmo que o resultado seja atribuído a uma variável de ponto flutuante.

No comando Math.trunc, a parte decimal é truncada (descartada). Para arredondar a divisão para o inteiro mais próximo, em vez de truncar o resultado, use o comando Math.round:

Divisão truncada:

let result = Math.trunc(5 / 2) // 2

Divisão arredondada:

let result = Math.round(5 / 2) // 3

Com o operador de resto da divisão, podemos saber se um número é múltiplo ou divisor de outro número. Por exemplo, para saber se uma variável é par ou ímpar, verificamos se seu resto é 0 ou 1, respectivamente:

par = 10
let impar = 15

let resto1 = par % 2
let resto2 = impar % 2

console.log(resto1, resto2)

Ao realizar divisões inteiras sucessivas por 10, obtendo o resto a cada divisão, conseguimos decompor um número em seus algarismos:

let num = 123;
let unidade = num % 10;
num = Math.trunc(num / 10);
let dezena = num % 10;
num = Math.trunc(num / 10);
let centena = num % 10;

console.log(centena, dezena, unidade)

Dia juliano

O dia juliano é a contagem contínua de dias desde o início do período juliano e é usado principalmente por astrônomos e em software para calcular facilmente os dias decorridos entre dois eventos.

O período juliano começou no dia juliano 0, que no calendário gregoriano proléptico, começou ao meio-dia de 24 de novembro de 4714 a.C.

Curiosidade: nessa data, três ciclos plurianuais começaram (os Ciclos Solar, Lunar e de Indicção), predecessores de qualquer data na história registrada.

Por exemplo, o dia juliano em 01/01/2000, ao meio dia do horário universal (UT), é 2451545.

Calendário Gregoriano

Antes de adotarmos o calendário gregoriano, usávamos o calendário juliano, que considera um ano bissexto a cada 4 anos. O problema desse calendário é que com o passar do tempo, ele relativamente fica atrasado em relação à volta da Terra ao Sol. Para corrigir esse atraso, em 05/10/1582, o Papa Gregório XIII acrescentou os 10 dias atrasados acumulados desde a adoção do calendário juliano e instituiu algumas outras regras para o ano bissexto. Essa reforma instituiu o calendário gregoriano, que começou a contar a partir de 15/10/1582, que equivale à data juliana de 05/10/1582. O calendário gregoriano proléptico [que ainda não aconteceu] se refere às datas anteriores a 15/10/1582, que são contadas retroativamente usando as regras de ano bissexto do calendário gregoriano.

Data gregoriana para dia juliano

Para realizar cálculos entre datas, consideramos apenas números inteiros, inclusive para os cocientes das divisões entre os números, descartando a parte decimal.

Para converter uma data gregoriana proléptica para seu dia juliano, a partir do dia 23 de novembro de 4714 a.C., usamos a seguinte fórmula:

J = (1461 * (Y + 4800 + (M - 14)/12))/4
  + (367 * (M - 2 - 12 * ((M - 14)/12)))/12
  - (3 * ((Y + 4900 + (M - 14)/12)/100))/4
  + D - 32075

Onde $J$ é o dia juliano, e $D$, $M$ e $Y$ são, respectivamente, o dia, mês e ano de uma data gregoriana.

Os meses de janeiro a dezembro são numerados de 1 a 12. Os dias dos meses são numerados de 1 a 31. Para o ano, é usada a numeração astronômica do ano, portanto, 1 AC é 0, 2 AC é -1 e 4714 AC é -4713. Use o dia anterior do mês se estiver tentando encontrar o dia juliano de um instante antes do meio-dia UT.

Observação: Eventos histórios anteriores a 15/10/1582 geralmente são datados usando o calendário juliano. Datas do calendário juliano precisam ser adaptadas para o calendário gregoriano. Por exemplo, o presidente americano George Washington nasceu em 11/02/1731 do calendário juliano, mas depois que a Inglaterra adotou o calendário gregoriano, em 1752, sua data de nascimento oficial foi alterada para 22/02/1732.

Dia juliano para dia da semana

Para saber o dia da semana a partir de um dia juliano, usamos a seguinte fórmula, começando do domingo como dia zero, sendo % o resto da divisão:

W1 = (J + 1) % 7

Mas se o horário for depois da meia-noite UT (e antes do meio-dia UT), então já está no próximo dia da semana.

Já considerando a segunda-feira como dia 1 e domingo como dia 7, usamos a seguinte fórmula:

Dia juliano para data gregoriana

Para realizar cálculos entre datas, consideramos apenas números inteiros, inclusive para os cocientes das divisões entre os números, descartando a parte decimal.

Para converter um dia juliano para sua data gregoriana, a partir do dia 0, usamos a seguinte fórmula:

f = J + 1401 + (4 * J + 274277) / 146097 * 3 / 4 - 38
e = 4 * f + 3
g = e % 1461 / 4
h = 5 * g + 2
D = h % 153 / 5 + 1
M = (h / 153 + 2) % 12 + 1
Y = e / 1461 - 4716 + (14 - M) / 12

Onde $J$ é o dia juliano, e $D$, $M$ e $Y$ são, respectivamente, o dia, mês e ano da data gregoriana.
