Caracteres e letras

O caráter é a base de todas as virtudes. – Confúcio.

Modelos matemáticos

Antes de entrar na pauta de caracteres, vamos falar um pouco sobre modelos matemáticos. O filósofo e matemático Pitágoras dizia que “tudo é número”. Essa frase traz a ideia de que podemos explicar a realidade e o mundo que nos cerca através de modelos matemáticos. Um modelo matemático é uma simplificação de um sistema real.

Usamos letras como símbolos que expressam sons, e assim, podemos nos comunicar através da escrita. E pelo simples fato de podermos quantificar e ordenar as letras do alfabeto, podemos imaginar que há matemática por detrás delas.

A propósito, um computador nada mais faz do que processar números, como seu nome indica. O processador, que executa todas as instruções de um programa, o faz através de sua Unidade Lógica Aritmética, responsável por realizar operações lógicas e aritméticas. Portanto, qualquer sistema computacional é um modelo matemático que traduz algum conceito da vida real. Por exemplo, qualquer cor pode ser formada pela combinação de três cores, vermelha (R), verde (G) e azul (B). Se atribuirmos uma escala de 0 a 255 (1 byte de dados) para cada uma das três cores, sendo 0 a total ausência de uma cor, e 255 a sua intensidade máxima, podemos representar numericamente (quase) qualquer cor. Assim, a cor roxa poderia ser representada pelos valores (127,0,127) para RGB. Esse é o modelo de cores RGB. Daí, para representar uma imagem, podemos ter uma sequência de valores RGB, cada uma formando um pixel da imagem. Claro, esse não é um modelo perfeito, pois a quantidade de cores existentes é infinita. Mas é o suficiente para muitas situações. Por isso que dizemos que um modelo matemático é uma simplificação, pois selecionamos algumas das infinitas variáveis que podem compor um sistema real, para modelá-lo conforme as limitações e necessidades que tivermos.

Tabela ASCII

No caso das letras, como nosso alfabeto possui 26 letras, poderíamos representar cada letra como um número, já que o computador só entende de números. As letras variariam de 1 a 26, sendo 1 a letra ‘a’, e 26 a letra ‘z’, por exemplo. Mas há também os sinais, como @ e #, e os próprios dígitos que compõem os números. A esses símbolos gráficos chamamos de caracteres.

Portanto, foi criado um modelo de representação de caracteres, chamado de tabela ASCII. Nela, podemos representar até 127 diferentes caracteres, que podemos dividir em quatro categorias:

  • Caracteres de controle (0–32 e 127): Os caracteres de controle não são visíveis. Eles são usados para enviar comandos ao PC ou à impressora. Com esses caracteres, você pode definir quebras de linha ou tabulações.
  • Caracteres Especiais (33–47 / 58–64 / 91–96 / 123–126): Caracteres especiais incluem todos os caracteres visíveis que não são letras nem números. Estes incluem sinais de pontuação ou caracteres técnicos e matemáticos.
  • Dígitos (30–39): Esses dígitos incluem os dez algarismos arábicos de 0 a 9.
  • Letras(65–90 / 97–122): As letras são divididas em dois blocos, com o primeiro grupo contendo as letras maiúsculas e o segundo grupo contendo as minúsculas.

Leia a tabela da seguinte forma: os números nas linhas representam a casa das dezenas, e os números nas colunas, a casa das unidades. Assim, a letra ‘A’ possui o código 65, e a letra ‘a’ o código 97.

Claro, há muito mais caracteres do que 127, como as letras do alfabeto cirílico (Д, Ђ, Э, Ю), e as letras acentuadas (à é î õ ü). Para representar mais caracteres, são usados outros modelos, como o sistema Unicode, que usa a tabela ASCII como base.

Programação

Um caractere (ou caracter) representa apenas um símbolo, como a letra 'e', o dígito '5', ou o símbolo '&'. Uma sequência de um ou mais caracteres é chamada de string. Diferentemente de outras linguagens de programação, o Javascript não possui um tipo de dados específico para caracteres, ele trata tudo como strings.

Alguns dos caracteres de controle são chamados de whitespaces e são usados para formatar o texto. São eles:

Código ASCII Código char Descrição
0 \0 Byte nulo. Usado para representar o final de uma string.
7 \a Apito.
8 \b Backspace.
9 \t Tabulação.
10 \n Quebra de linha.
11 \v Tabulação vertical.
12 \f Fim de página.
13 \r Retorno de carro.

Os whitespaces também são conhecidos como sequências de escape. Eles sempre começam com uma barra invertida, e o caractere seguinte determina a interpretação da sequência de escape. Tanto a barra invertida quanto as aspas têm significados especiais em uma string. Portanto, para representar esses próprios caracteres, você deve prefixá-los com uma barra invertida:

console.log('Essa é a barra invertida: "\\".')
console.log('Uma palavra entre aspas simples: \'javascript\'.')
console.log("Uma palavra entre aspas duplas: \"javascript\".")

Observe que quando você usa uma string delimitada por aspas simples, não precisa prefixar as aspas duplas para incluí-las na string, e vice-versa.

Você pode saber o código de um caractere usando a função String.fromCharCode. E também pode saber o caractere correspondente a um código usando a função charCodeAt:

let codigo_a = 97
// convertendo de código numérico para letra
let letra_a = String.fromCharCode(codigo_a)

console.log("\nLetra:", letra_a)
console.log("Código numérico:", codigo_a)

let letra_j = 'j'
// convertendo de letra para código numérico
let codigo_j = letra_j.charCodeAt()

console.log("\nLetra:", letra_j)
console.log("Código numérico:", codigo_j)

Veja outro exemplo:

let a = 97
let U = 85
let cifrao = 36
let maior = ">"
let arroba = "@"
let dois = "2"

console.log(a, String.fromCharCode(a))
console.log(U, String.fromCharCode(U))
console.log(cifrao, String.fromCharCode(cifrao))
console.log(maior, maior.charCodeAt())
console.log(arroba, arroba.charCodeAt())
console.log(dois, dois.charCodeAt())

No Javascript não é possível realizar operações aritméticas entre números e caracteres, mas se você combinar as funções String.fromCharCode e charCodeAt, consegue fazer coisas interessantes:

let c1 = "d"
let c2 = "h"
let n1 = 7
 
let c3 = String.fromCharCode(n1 + c1.charCodeAt())
console.log(n1, "letras depois de", c1, "é", c3)
 
let n2 = c2.charCodeAt() - c1.charCodeAt() - 1
console.log("há", n2, "letras entre", c1, "e", c2)
 
let c4 = c1.charCodeAt() * c2.charCodeAt()
console.log(c1, "*", c2, "=", c4)
 
let n3 = c1.charCodeAt() + c2.charCodeAt()
console.log(c1, "+", c2, "=", n3)

Formatando texto

Textos são representados entre aspas no Javascript. O Javascript não faz distinção entre aspas simples e duplas, mas é importante usar a mesma aspa para abrir e fechar um texto. Além disso, para manter o código legível, é aconselhável seguir um padrão consistente.

Existem maneiras de formatar textos para alterar sua disposição. Já vimos alguns caracteres de controle conhecidos como whitespaces, que são capazes de modificar a formatação do texto.

Frequentemente, quando desejamos apresentar um texto que inclui valores de variáveis, é mais prático usar placeholders (marcadores de posição), que indicam onde o valor de uma variável deve ser inserido no texto. Esses placeholders também são chamados de especificadores de formato ou caracteres de conversão, pois permitem personalizar a maneira como um valor é exibido. Para utilizar placeholders em strings, você deve utilizar acentos graves em vez de aspas . Em seguida, coloque a variável precedida por um cifrão e entre chaves nas posições onde deseja inserir seu valor:

let inteiro = 10
let decimal = 12.34
let letra = 'A'

console.log(`Um inteiro: ${inteiro}.\nUm decimal: ${decimal}.\nUma letra: ${letra}`)

Na verdade, é possível incluir qualquer expressão válida dentro de um placeholder:

let a = 65
let b = 2

console.log(`A soma de ${a} e ${b} é ${a+b}.\nO caractere correspondente a 65 + 2 é ${String.fromCharCode(a+b)}`)

Uma expressão é uma combinação de variáveis, operadores, constantes e funções que resulta em um valor.

Assim como em uma string delimitada por aspas é necessário prefixar caracteres especiais com uma barra invertida poder exibi-los, em uma string delimitada por acentos graves, é necessário prefixar o cifrão e a chave:

let num = 10;
console.log(`Um número dentro de chaves duplas: \${ ${num} }.`)

Exercícios

  1. Dado um caractere, qual o seu código na tabela ASCII?
  2. Dado um caractere, ele é uma letra?
  3. Dada uma letra, qual a sua posição no alfabeto, sendo ‘a’ a posição 1 e ‘z’ a posição 26?
  4. Dada uma letra, ela é maiúscula ou minúscula?
  5. Dada uma letra, ela é vogal ou consoante?
  6. Dado um caractere, ele é um algarismo?
  7. Dada uma letra maiúscula, qual sua correspondente em minúsculo?
  8. Dada uma letra e um inteiro N, qual a letra correspondente ao andar N casas à frente da letra lida (ou atrás, se o N for negativo).
  9. Dadas duas letras, quantas letras há entre as duas?
