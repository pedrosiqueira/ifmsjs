Aleatoriedade

Sorte é o encontro do preparo com a oportunidade.

Considere o seguinte trecho de código para gerar números aleatórios. Preste atenção aos comentários para entender o que está se passando.

// Sorteia um float no intervalo [0,1)
let sorteado = Math.random();
console.log(sorteado);

// Transforma em um float entre [0,10)
sorteado *= 10;
console.log(sorteado);

// Descarta a parte decimal
let int_sorteado = Math.trunc(sorteado);
console.log(int_sorteado);

A função Math.random gera um float no intervalo [0,1). A notação matemática [0,1) significa o intervalo de números entre o 0 inclusivo e o 1 exclusivo, isto é, números maiores ou iguais a 0 e menores que 1.

O trecho de código anterior gerou um inteiro aleatório entre [0,10). Podemos simplificar em apenas uma instrução a geração de um inteiro aleatório entre [0,n):

let n = 20
let sorteado = Math.trunc(Math.random() * n)
console.log(sorteado)

Se você não quiser começar do zero, pode usar um valor de deslocamento. Por exemplo, para gerar um número aleatório entre [10,20], você pode fazer:

let primeiro = 10
let ultimo = 20
let quantidade = ultimo - primeiro + 1
let sorteado = primeiro + Math.trunc(Math.random() * quantidade)
console.log(sorteado)

Você também pode gerar um número aleatório dentro de um intervalo discreto. Por exemplo, para gerar um múltiplo de 5 entre [50,100], você pode fazer:

let primeiro = 50
let ultimo = 100
let diferenca = 5
let quantidade = (ultimo - primeiro) / diferenca + 1
let sorteado = primeiro + diferenca * Math.trunc(Math.random() * quantidade)
console.log(sorteado)

Essa é a fórmula geral para gerar números inteiros. Você pode ajustá-la conforme sua necessidade. Apenas altere as variáveis primeiro, ultimo e diferenca. Elas representam, respectivamente, o primeiro número inclusivo do intervalo, o último número inclusivo do intervalo, e a diferença entre dois números consecutivos do intervalo.

Exercícios

  1. Dado um inteiro $n$, gerar três inteiros aleatórios entre 0 e $n$.
  2. Dado um inteiro $n$, gerar três inteiros aleatórios entre $-n$ e $n$.
  3. Dados dois inteiros $m$ e $n$, gerar três inteiros aleatórios entre $m$ e $n$.
  4. Dados dois inteiros $m$ e $n$, gerar três inteiros aleatórios múltiplos de $m$ entre $0$ e $n$.
  5. Gerar dois inteiros aleatórios cuja soma esteja entre [0 e 100).
  6. Gerar uma letra aleatória entre 'a' e 'z'.
  7. Gerar uma senha aleatória de 8 caracteres com letras maiúsculas, minúsculas e dígitos.
  8. Gerar três decimais aleatórios entre [-100 e 100], com até duas casas decimais.
  9. Ler um número inteiro e escrever um valor aleatório com base nesse número: uma letra maiúscula se o usuário inseriu 1, uma letra minúscula se o usuário inseriu 2 e um dígito se o usuário inseriu qualquer valor que não seja 1 ou 2.
