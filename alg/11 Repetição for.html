Repetição for

A roda já deu uma volta completa. — William Shakespeare

Principios da repetição controlada por contador

Na repetição controlada por contador, sabemos de antemão quantas vezes o corpo da repetição será executado. O contador começa de um valor inicial e é atualizado a cada iteração (ciclo) até alcançar um valor final.

Em vez de usarmos a instrução while para repetição por contador, podemos usar a instrução for. A instrução for especifica no seu cabeçalho todos os detalhes da repetição.

// instruções antes do for

for (início; fim; atualização) {
    // instruções do corpo do for
}

// instruções depois do for

O início especifica o valor inicial do contador. É executado apenas uma vez, antes da primeira iteração.

O fim especifica o valor final do contador. É uma condição, que é executada no início de cada iteração. Se a condição é verdadeira, as instruções do corpo do for são executadas, senão, o for é encerrado e o programa continua da próxima instrução depois do corpo do for.

E atualização especifica o próximo valor do contador. É executada no final de cada iteração. Ela modifica o contador para que uma hora a condição encerre o for. Terminada a atualização, o programa volta para o início da próxima iteração.

Veja um exemplo do for ao lado do while para comparação.

let inicio = 1
let fim = 11
console.log("Contando até 10...")

let cont = inicio // valor inicial
while (cont < fim) { // valor final
    console.log(cont)
    cont++ // atualização
}

console.log("Fim!")
console.log("Valor final:", cont)
 
let inicio = 1
let fim = 11
console.log("Contando até 10...")


for (let cont = inicio; cont < fim; cont++) {
    console.log(cont)
}


console.log("Fim!")
console.log("Valor final:", cont)

As expressões em um cabeçalho for são opcionais.

  • Você poderia omitir a inicialização se o contador é inicializado antes do for.
  • Você poderia omitir a atualização se o contador é atualizado dentro do for.
  • Você poderia omitir a condição se quisesse manter o controle da repetição dentro do corpo do for (veremos isso mais adiante com os comandos continue/break).

Assim como qualquer estrutura que tem um corpo, as chaves do for são opcionais quando o seu corpo possui apenas uma instrução. Mesmo assim, é recomendável sempre colocar chaves para explicitar o corpo do for.

Os dois pontos-e-vírgulas no cabeçalho do for são obrigatórios. Mas atenção para não colocar um ponto-e-vírgula depois dos parênteses do for!

Variáveis declaradas dentro do for, inclusive variáveis do cabeçalho, só existem enquanto o for estiver sendo executado.

Observação: Assim como o while, o for pode nunca ser executado! Se a condição é falsa durante a primeira vez, o corpo do for não é executado.

Diferenças e semelhanças entre for e while

Tudo o que dá para fazer usando while também dá usando for e vice-versa. Porém dependendo da situação, um pode ser mais adequado que outro. Em geral, as instruções for são utilizadas para repetição controlada por contador: quando sabemos quantas vezes queremos repetir determinado trecho de código… E as instruções while, para repetição controlada por sentinela (flag): quando não sabemos o número de vezes que queremos repetir determinado trecho de código.

let i = 1
let soma = 0
while (i < 5) {
    console.log("+", i)
    soma += i
    i++
}
console.log("= ", soma)
 

let soma = 0
for (let i = 1; i < 5; i++) {
    console.log("+", i, end=" ")
    soma += i
}

console.log("= ", soma)

Mais possibilidades com o for

O contador pode ser decrementado:

console.log("Contagem regressiva:");
for (let i = 10; i > 0; i--) {
    console.log(i, "...");
}
console.log("Decolar!");

O contador não precisa ser atualizado apenas de um em um:

console.log("todos os múltiplos de três entre zero e cem:");
for (let n = 0; n <= 100; n += 3) {
    console.log(n);
}

Sugira uma correção para o seguinte erro:

let i = 5;
for (let j = 3; j < 17; j += 4) {
    console.log(i, j);
    i--;
}
console.log(i, j); // erro: j não existe aqui

O início e a atualização podem ter múltiplas expressões separadas por vírgula:

for (let i = 1, j = 10; i < j; i++, j--) {
    console.log(i, j);
}

Como essas expressões podem tornar o código mais confuso, não é comum fazer assim. Outro exemplo:

let bit, p;
for(bit=512, p=0; bit > 1; bit/=2, p+=1){
    console.log(bit);
}
console.log("=", p);

Na verdade, atualização pode ser qualquer expressão válida:

for (let i = 1; i < 1025; console.log(i *= 2));

O ponto-e-vírgula no final do for indica que seu corpo é vazio. Na atualização, ao mesmo tempo que i é dobrado, também é exibido. Apesar de funcionar, essa estrutura não é recomendada por ser confusa.

Mais exemplos de for

Contando de 1 a 100 em incrementos de 1.

for (let i = 1; i <= 100; i++) { … }

Contando de 100 a 1 em decrementos de 1.

for (let i = 100; i >= 1; i--) { … }

Contando de 7 a 77 em incrementos de 7.

for (let i = 7; i <= 77; i += 7) { … }

Contando de 20 a 10 em decrementos de 2.

for (let i = 20; i >= 10; i -= 2) { … }

Incrementando entre os valores 2, 5, 8, 11, 14, 17, 20.

for (let i = 2; i <= 20; i += 3) { … }

Decrementando entre os valores 99, 88, 77, 66, 55, 44, 33, 22, 11, 0.

for (let i = 99; i >= 0; i -= 11) { … }

Laços aninhados

Seja estruturas de seleção ou de repetição, podemos colocá-las umas dentro das outras.

Veja esse exemplo de if-else dentro de for. Qual a saída?

let i = 12
for (let j = 1; j <= i; j++) {
    console.log(j)
    if (j % 5 == 0 || j % 3 == 0)
        console.log("A")
    if (i % j == 0)
        console.log("B")
    else
        console.log("C")
}

Vamos examinar um exemplo de laço for simples:

for (let j = 0; j < 5; j++)
    console.log(j)

A linha 2 é executada 5 vezes. Agora, vamos colocar esse laço dentro de outro laço:

for (let i = 0; i < 3; i++) {
    console.log()
    for (let j = 0; j < 5; j++) {
        console.log(j)
    }
}

Neste novo código, devido ao laço da linha 1, as linhas 2 a 5 são executadas 3 vezes. A linha 4, que já era executada 5 vezes no laço interno, agora é executada um total de 3 * 5 = 15 vezes!

O laço interno repete a mesma quantidade de vezes a cada iteração (ciclo) do laço externo. Mas poderíamos alterar esse comportamento:

for (let i = 0; i < 3; i++) {
    console.log()
    for (let j = i; j < 5; j++) {
        console.log(j)
    }
}

Qual a saída do seguinte código?

let externo, interno;
console.log("Início");
for (externo = 1; externo <= 3; externo++) {
    console.log(">");
    for (interno = 1; interno <= 5; interno++) {
        console.log(externo, interno);
    }
    console.log("<");
}
console.log("Fim");

Qual a saída do seguinte código?

for (let i = 0; i < 3; i++) {
    for (let j = 0; j < 4; j++) {
        for (let k = 0; k < 2; k++) {
            console.log(3)
        }
        console.log(2)
    }
    console.log(1)
}

Por exemplo, ao investir $\$1000$ em um ativo que rende $5\%$ ao ano, ao final de 10 anos você terá acumulado $M=\$1000(1+0.05)^{10}=\$1628.89$.
