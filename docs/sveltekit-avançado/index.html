Sites Permitidos para Aulas de Desenvolvimento Web

  • pedrosiqueira.github.io

    Nosso site com o conteúdo da matéria e muito mais!

  • getbootstrap.com

    O site oficial do Bootstrap, o framework de front-end mais popular do mundo. Ele fornece componentes CSS e JavaScript prontos para uso, permitindo a criação rápida de sites responsivos e com design moderno.

  • stackoverflow.com

    Uma comunidade online de perguntas e respostas para programadores. É um recurso indispensável para solucionar problemas de codificação, aprender com as dúvidas de outros desenvolvedores e compartilhar conhecimento.

  • svelte.dev

    O site oficial do Svelte, um framework JavaScript moderno para a construção de interfaces de usuário. É conhecido por sua abordagem de compilação que resulta em um código mais rápido e eficiente.

  • developer.mozilla.org

    A MDN Web Docs (Mozilla Developer Network) é uma fonte abrangente de documentação para tecnologias web, incluindo HTML, CSS e JavaScript. É um recurso essencial para desenvolvedores web de todos os níveis.

  • w3schools.com

    Um popular site de tutoriais e referências para diversas tecnologias web. Oferece exemplos de código interativos que permitem aos alunos experimentar diretamente no navegador.

  • cssportal.com

    Um portal com diversas ferramentas e geradores de código CSS. É útil para criar layouts, gradientes, sombras e outros efeitos visuais, além de oferecer um editor de CSS online.

  • translate.google.com

    O Google Tradutor é uma ferramenta de tradução automática que pode ser útil para traduzir documentações e artigos técnicos, facilitando o acesso a conteúdos em diferentes idiomas.

  • deckofcardsapi.com

    Uma API RESTful gratuita que permite aos desenvolvedores simular um baralho de cartas. É um excelente recurso para praticar requisições HTTP e manipulação de dados em projetos de programação.

  • jsonplaceholder.typicode.com

    Um serviço de API REST online e gratuito, ideal para testes e prototipagem. Fornece dados falsos (placeholders) no formato JSON, o que é perfeito para simular interações com um back-end.

  • dummyjson.com

    Oferece uma variedade de endpoints REST com dados JSON falsos para diversos fins, como produtos, usuários e posts. É uma ferramenta muito útil para o desenvolvimento de front-end sem a necessidade de um back-end funcional.
