  1. Considere o seguinte arquivo src/routes/p2/1/+page.svelte:
    2. <script>
  let { data } = $props();

<form>
  <input name="a" type="number" />
  <input name="b" type="number" />
  <button>Vai!</button>

{#if data.status == 400}
  <p>{data.erro}</p>
{:else}
  <p>Bem-vindo!</p>
{/if}
  2. Considere também o seguinte arquivo src/routes/p2/1/+page.server.js:
    3. import { redirect } from '@sveltejs/kit';

export async function load({ url }) {
    const var1 = url.searchParams.get('a');
    const var2 = url.searchParams.get('b');

    if (var1 == var2) return { status: 400, erro: "Os valores não podem ser iguais!" };

    redirect(303, `/p2/2?x=${var1}&y=${var2}`);
}
  3. Implemente a rota em src/routes/p2/2 que exibe os parâmetros de consulta que foram enviados pela rota em src/routes/p2/1 através do redirecionamento.
  4. Modifique os arquivos em src/routes/p2/1 de modo a usar requisição POST em vez de requisição GET.
