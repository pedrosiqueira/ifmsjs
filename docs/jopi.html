Exercícios

Considere a API REST dummyjson.

  1. Implemente a rota /p2/products/[category]: liste os produtos da API cuja categoria corresponda ao valor do parâmetro de rota [category].
  2. Exiba os produtos em formato de cards. Cada card deve mostrar o título, preço, categoria e imagem do produto.

Dicas:

  • A propriedade thumbnail de cada produto contém a imagem do produto.
  • A propriedade price contém o preço do produto.
  • A propriedade title contém o título do produto.
  • A propriedade category contém a categoria do produto.
