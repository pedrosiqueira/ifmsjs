Exercício

Implemente uma janela de chat fictício com os seguintes requisitos:

  • No topo, exiba o nome de um contato fictício.
  • No meio, mostre as mensagens trocadas na conversa.
  • Cada mensagem deve exibir o rementente, o conteúdo e a data em que foi enviada.
  • As mensagens do usuário devem aparecer alinhadas à direita.
  • Na parte inferior, inclua um campo de entrada para digitar novas mensagens e um botão de envio.

Quando o usuário clicar no botão ou pressionar Enter:

  • A mensagem digitada deve aparecer ao final da conversa, acompanhada da data e hora em que foi enviada.

No momento, não é necessário implementar o envio de mensagens pelo contato fictício; o chat deve apenas exibir as mensagens enviadas pelo usuário.

Dicas:

  • Utilize os cards do Bootstrap para estilizar cada mensagem.
  • Para armazenar a data e hora atual, use: let data = new Date().
  • Cada mensagem pode ser representada como um objeto com três propriedades: remetente, conteudo e data.
  • Para exibir a data de forma legível, use: data.toLocaleString().
