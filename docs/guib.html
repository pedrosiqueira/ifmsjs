Exerccio

Implemente uma janela de chat fictcio com os seguintes requisitos:

  • No topo, exiba o nome de um contato fictcio.
  • No meio, mostre as mensagens trocadas na conversa.
  • Cada mensagem deve exibir o rementente, o contedo e a data em que foi enviada.
  • As mensagens do usurio devem aparecer alinhadas direita.
  • Na parte inferior, inclua um campo de entrada para digitar novas mensagens e um boto de envio.

Quando o usurio clicar no boto ou pressionar Enter:

  • A mensagem digitada deve aparecer ao final da conversa, acompanhada da data e hora em que foi enviada.

No momento, no necessrio implementar o envio de mensagens pelo contato fictcio; o chat deve apenas exibir as mensagens enviadas pelo usurio.

Dicas:

  • Utilize os cards do Bootstrap para estilizar cada mensagem.
  • Para armazenar a data e hora atual, use: let data = new Date().
  • Cada mensagem pode ser representada como um objeto com trs propriedades: remetente, conteudo e data.
  • Para exibir a data de forma legvel, use: data.toLocaleString().
