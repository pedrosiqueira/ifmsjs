Adicionando Bootstrap ao SvelteKit

Para utilizar SvelteKit com Bootstrap, modifique o arquivo src/app.html (em verde as modificações, em azul os acréscimos):

<!doctype html>
<html lang="pt">

<head>
    <meta charset="utf-8" />
    <link rel="icon" href="%sveltekit.assets%/favicon.png" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
    %sveltekit.head%
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH" crossorigin="anonymous">
</head>

<body data-sveltekit-preload-data="hover">
    <div style="display: contents">%sveltekit.body%</div>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js" integrity="sha384-YvpcrYf0tY3lHB60NNkmXc5s9fDVZLESaAA55NDzOxhy9GkcIdslK1eN7N6jIeHz" crossorigin="anonymous"></script>
</body>

</html>

Na linha 2 alteramos nosso aplicativo para ser reconhecido pelos navegadores como de idioma português. Nas linhas 9 e 14 adicionamos o css e o js do bootstrap.

Nossas primeiras páginas

Crie o arquivo src/routes/sobre/+page.svelte com informações sobre teu site. Também modifique o arquivo src/routes/+page.svelte com uma mensagem de boas vindas personalizada para o teu site.

Adicionando uma barra de navegação

Vimos no tutorial do sveltekit como fazer uma barra de navegação personalizada. Mas aqui, utilizaremos a barra de navegação do Bootstrap. Crie o arquivo src/routes/+layout.svelte:

<script>
    let { children } = $props();
</script>

<nav class="navbar navbar-expand-sm bg-body-tertiary">
    <div class="container-fluid">
        <a class="navbar-brand" href="/#">DW4</a>
        <button
            class="navbar-toggler"
            type="button"
            data-bs-toggle="collapse"
            data-bs-target="#navbarSupportedContent"
            aria-controls="navbarSupportedContent"
            aria-expanded="false"
            aria-label="Toggle navigation"
        >
            <span class="navbar-toggler-icon"></span>
        </button>
        <div class="collapse navbar-collapse" id="navbarSupportedContent">
            <ul class="navbar-nav">
                <li class="nav-item">
                    <a class="nav-link" href="/sobre">Sobre</a>
                </li>
            </ul>
        </div>
    </div>
</nav>

{@render children()}

Explicação:

  • Linha 7: O botão da marca do teu site. É aqui que geralmente fica o logotipo do teu site.
  • Linhas 8-18: Botão que expande os itens do teu navbar quando a tela fica pequena.
  • Linha 20: Os itens do teu navbar. No momento, só temos um, que é a página "Sobre".

Estilizamos nossos elementos com Bootstrap através do atributo class. Explicação das principais classes usadas aqui:

  • navbar: Estiliza o elemento nav.
  • navbar-expand-sm Especifica que os itens dentro do div com classe collapse navbar-collapse serão ocultados em telas pequenas. As opções de ocultamento são: .navbar-expand{-sm|-md|-lg|-xl|-xxl}, que significam, respectivamente, pequeno, médio, grande, extra grande, extra extra grande.
  • bg-body-tertiary: Define a cor de fundo do navbar como um cinza suave. O Bootstrap vem com uma paleta de cores padrão para o fundo. Experimente mudar a cor de fundo do teu navbar com uma das opções do Bootstrap!
  • container-fluid: Define que esse div ocupará 100% da largura da tela. Outras opções de expansão da largura de divs podem ser encontradas aqui.
