Preparando o ambiente de desenvolvimento

Instalação

Vamos usar duas coisas para desenvolver nossos aplicativos: (1) O VSCode, onde vamos escrever nossos códigos, e (2) O NodeJS, que vai executar nossos códigos.

Para instalar o NodeJS, siga os passos da seção Instalando um ambiente de execução Javascript. Depois de instalado o NodeJS, execute pelo terminal o comando npm install -g pnpm.

Para instalar o VSCode, siga os passos da seção Ambiente de desenvolvimento integrado.

Criando e executando nosso projeto

Se você quiser configurar seu projeto do zero, siga as etapas da Alternativa 1. Se quiser baixar o projeto já configurado, siga as etapas da Alternativa 2.

Criando...

Alternativa 1 - Criando o projeto do zero

Abra o terminal onde quer criar o projeto e execute os seguintes comandos:

pnpm create svelte@latest dw4
cd dw4
pnpm install
code -r .

Quando solicitado, escolha as seguintes opções:

    ┌  Welcome to SvelteKit!
    │
    ◇  Which Svelte app template?
    │  Skeleton project
    │
    ◇  Add type checking with TypeScript?
    │  No
    │
    ◇  Select additional options (use arrow keys/space bar)
    │  Add ESLint for code linting, Add Prettier for code formatting
    │
    └  Your project is ready!

Esses comandos fazem respectivamente: (1) cria um projeto sveltekit com o nome dw4; (2) abre a pasta do projeto; (3) configura o projeto; e (4) abre a pasta do projeto pelo vscode.

Alternativa 2 - Baixando o projeto já configurado

Se você já seguiu a Alternativa 1, pode pular a Alternativa 2...

  1. Baixe o projeto do link https://github.com/pedrosiqueira/dw4/archive/refs/heads/main.zip.
  2. O arquivo baixado está compactado em .zip. Descompacte o arquivo no local onde deseja armazenar seu projeto.
  3. Abra o projeto no vscode (atalho Ctrl+k Ctrl+o).
  4. Com o terminal na pasta do projeto, execute pnpm install.

Executando...

Após criar o seu projeto, com o terminal na pasta do projeto, execute o comando pnpm run dev -- --open. Se ocorreu tudo bem, será aberta uma página no navegador web com seu aplicativo web rodando!
