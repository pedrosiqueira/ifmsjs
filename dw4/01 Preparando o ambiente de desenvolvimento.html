Criando e executando nosso projeto

Se você tem o git e quiser clonar o projeto pelo git, execute o script da Alternativa 0 ou siga as etapas da Alternativa 1, que faz manualmente o mesmo que a Alternativa 0. Se quiser baixar o projeto já configurado, siga as etapas da Alternativa 2. Se você quiser configurar teu projeto do zero, siga as etapas da Alternativa 3.

Alternativa 0 - Mamão com açúcar

Execute o seguinte código no terminal:

Invoke-Expression ((New-Object System.Net.WebClient).DownloadString('https://raw.githubusercontent.com/pedrosiqueira/ifmsjs/refs/heads/main/scripts/03_dw4_project_setup.ps1'))

Alternativa 1 - Clonando o projeto pelo git

  1. Copie este link: https://github.com/pedrosiqueira/dw4.git.
  2. No vscode, tecle F1, digite "clone" e selecione a opção "Git: Clone".
  3. Na caixa de entrada que aparece, cole o link copiado anteriormente e tecle Enter.
  4. Escolha um local para o seu projeto e abra-o no vscode.
  5. Com o terminal na pasta do projeto, execute pnpm install.

Alternativa 2 - Baixando o projeto já configurado

  1. Baixe o projeto do link https://github.com/pedrosiqueira/dw4/archive/refs/heads/main.zip.
  2. O arquivo baixado está compactado em .zip. Descompacte o arquivo no local onde deseja armazenar seu projeto.
  3. Abra o projeto no vscode (atalho Ctrl+k Ctrl+o).
  4. Com o terminal na pasta do projeto, execute pnpm install.

Alternativa 3 - Criando o projeto do zero

Abra o terminal onde quer criar o projeto e execute os seguintes comandos:

pnpm dlx sv create dw4
cd dw4
pnpm install
code -r .

Esses comandos fazem respectivamente: (1) cria um projeto sveltekit com o nome dw4; (2) abre a pasta do projeto; (3) configura o projeto; e (4) abre a pasta do projeto pelo vscode.

Executando o projeto

Após criar teu projeto, com o terminal aberto na pasta do mesmo, execute o comando pnpm run dev -- --open. Se ocorreu tudo bem, uma página será aberta automaticamente no navegador web, exibindo seu aplicativo web em execução. Para encerrar o aplicativo, basta voltar ao terminal onde ele está rodando e pressionar Ctrl+C.

Se você possuir permissões administrativas, é possível disponibilizar o site para todos os dispositivos da sua rede local utilizando o comando pnpm run dev -- --host 0.0.0.0. Após isso, tente acessar o site pelo teu celular ou outro dispositivo, utilizando o endereço IP exibido no terminal.

Caso deseje expor o site para a Internet, é possível, mas esse processo está fora do escopo desta aula. Isso demandaria configurações adicionais, como mexer no modem, além de apresentar riscos de segurança, já que seu computador ficaria acessível publicamente na rede.
