Guia Rápido do SQLite com os principais comandos para referência rápida

Este guia cobre os comandos essenciais do SQLite para interagir com o banco de dados, criar tabelas, manipular dados e utilizar funcionalidades-chave do SQL.

Linguagem de Definição de Dados (DDL)

Criar Banco de Dados

O SQLite cria automaticamente um banco de dados ao tentar abri-lo.

SQLite Drizzle ORM 

sqlite3 meubanco.db

Criar Tabela

Cria uma nova tabela com colunas e tipos de dados.

SQLite Drizzle ORM 

CREATE TABLE umatabela (
    um_numero INTEGER,
    um_decimal REAL,
    um_texto TEXT,
    um_arquivo BLOB
);
 
import { sqliteTable, integer, real, text, blob } from 'drizzle-orm/sqlite-core';

export const umaTabela = sqliteTable('umatabela', {
  um_numero: integer(),
  um_decimal: real(),
  um_texto: text(),
  um_arquivo: blob(),
});

Alterar Tabela

Adicionar uma coluna a uma tabela existente.

SQLite Drizzle ORM 
ALTER TABLE umatabela ADD COLUMN id INTEGER;
 
import { sql } from 'drizzle-orm';
import { umaTabela } from './schema'; // Importe a tabela existente

await db.execute(sql`ALTER TABLE ${umaTabela} ADD COLUMN id INTEGER;`);

Adicionar uma chave estrangeira a uma tabela existente.

SQLite Drizzle ORM 
ALTER TABLE umatabela ADD COLUMN umatabela_id INTEGER REFERENCES umatabela(id);
 
await db.execute(
  sql`ALTER TABLE ${umaTabela} ADD COLUMN umatabela_id INTEGER REFERENCES ${umaTabela}(id);`
);

Excluir Tabela

Remove uma tabela do banco de dados.

SQLite Drizzle ORM 
DROP TABLE umatabela;
 
await db.execute(sql`DROP TABLE ${umaTabela};`);

Restrições no SQLite

Chave Primária

Define uma chave primária para uma coluna.

SQLite Drizzle ORM 
CREATE TABLE categoria (
    rotulo TEXT PRIMARY KEY
);
 
export const categoria = sqliteTable('categoria', {
  rotulo: text().primaryKey(),
});

Autoincremento

Usar AUTOINCREMENT com INTEGER PRIMARY KEY para gerar valores únicos automaticamente.

SQLite Drizzle ORM 
CREATE TABLE usuarios (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    nome_usuario TEXT NOT NULL,
    email TEXT NOT NULL UNIQUE,
    data_nascimento TEXT,
    criado_em TEXT DEFAULT CURRENT_TIMESTAMP
);
 
export const usuarios = sqliteTable('usuarios', {
  id: integer().primaryKey({ autoIncrement: true }),
  nome_usuario: text().notNull(),
  email: text().notNull().unique(),
  data_nascimento: text(),
  criado_em: text().default(sql`CURRENT_TIMESTAMP`),
});

Restrição Única

Garante que todos os valores de uma coluna sejam únicos.

SQLite Drizzle ORM 
CREATE TABLE usuarios (
    id INTEGER PRIMARY KEY,
    email TEXT UNIQUE
);
 
export const usuarios = sqliteTable('usuarios', {
  id: integer('id').primaryKey(),
  email: text('email').unique(),
});

Restrição NOT NULL

Garante que uma coluna não possa ter valores NULL.

SQLite Drizzle ORM 
CREATE TABLE usuarios (
    id INTEGER PRIMARY KEY,
    nome_usuario TEXT NOT NULL
);
 
export const usuarios = sqliteTable('usuarios', {
  id: integer('id').primaryKey(),
  nomeUsuario: text('nome_usuario').notNull(),
});

Chave Estrangeira

Define uma chave estrangeira para impor relacionamentos entre tabelas.

SQLite Drizzle ORM 
CREATE TABLE posts (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    titulo TEXT NOT NULL,
    conteudo TEXT NOT NULL,
    usuario_id INTEGER NOT NULL,
    criado_em TEXT DEFAULT CURRENT_TIMESTAMP,
    atualizado_em TEXT DEFAULT CURRENT_TIMESTAMP CHECK(atualizado_em >= criado_em),
    FOREIGN KEY (usuario_id) REFERENCES usuarios(id)
);
 
export const posts = sqliteTable('posts', {
  id: integer('id').primaryKey({ autoIncrement: true }),
  titulo: text('titulo').notNull(),
  conteudo: text('conteudo').notNull(),
  usuarioId: integer('usuario_id')
    .notNull()
    .references(() => usuarios.id),
  criadoEm: text('criado_em').default(sql`CURRENT_TIMESTAMP`),
  atualizadoEm: text('atualizado_em')
    .default(sql`CURRENT_TIMESTAMP`)
    .check(sql`atualizado_em >= criado_em`),
});

Restrição CHECK

Adiciona uma validação de dados na entrada.

SQLite Drizzle ORM 
CREATE TABLE posts (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    atualizado_em TEXT DEFAULT CURRENT_TIMESTAMP CHECK(atualizado_em >= criado_em)
);
 
export const posts = sqliteTable('posts', {
  id: integer('id').primaryKey({ autoIncrement: true }),
  atualizadoEm: text('atualizado_em')
    .default(sql`CURRENT_TIMESTAMP`)
    .check(sql`atualizado_em >= criado_em`),
});

Linguagem de Manipulação de Dados (DML)

Inserir Dados

Inserir um novo registro em uma tabela.

SQLite Drizzle ORM 
INSERT INTO usuarios (nome_usuario, email)
VALUES ('maria_silva', 'maria@silva.com.br');
 
await db.insert(usuarios).values({
  nomeUsuario: 'maria_silva',
  email: 'maria@silva.com.br',
});

Inserir vários registros de uma vez.

SQLite Drizzle ORM 
INSERT INTO usuarios (nome_usuario, email, data_nascimento)
VALUES
    ('francisca_pereira', 'francisca@pereira.com', '2002-11-05'),
    ('jose_santos', 'jose@santos.com', '2004-02-29'),
    ('joao_souza', 'joao@souza.com', '2001-12-31'),
    ('ana_oliveira', 'ana@oliveira.com', '2002-11-05');

INSERT INTO posts (titulo, conteudo, usuario_id)
VALUES
    ('primeiro artigo', 'conteúdo do primeiro artigo', 1),
    ('segundo artigo', 'conteúdo do segundo artigo', 2),
    ('terceiro artigo', 'conteúdo do terceiro artigo', 2),
    ('quarto artigo', 'conteúdo do quarto artigo', 1),
    ('quinto artigo', 'conteúdo do quinto artigo', 2);
 
await db.insert(usuarios).values([
  { nomeUsuario: 'francisca_pereira', email: 'francisca@pereira.com', dataNascimento: '2002-11-05' },
  { nomeUsuario: 'jose_santos', email: 'jose@santos.com', dataNascimento: '2004-02-29' },
  { nomeUsuario: 'joao_souza', email: 'joao@souza.com', dataNascimento: '2001-12-31' },
  { nomeUsuario: 'ana_oliveira', email: 'ana@oliveira.com', dataNascimento: '2002-11-05' },
]);

await db.insert(posts).values([
  { titulo: 'primeiro artigo', conteudo: 'conteúdo do primeiro artigo', usuarioId: 1 },
  { titulo: 'segundo artigo', conteudo: 'conteúdo do segundo artigo', usuarioId: 2 },
  { titulo: 'terceiro artigo', conteudo: 'conteúdo do terceiro artigo', usuarioId: 2 },
  { titulo: 'quarto artigo', conteudo: 'conteúdo do quarto artigo', usuarioId: 1 },
  { titulo: 'quinto artigo', conteudo: 'conteúdo do quinto artigo', usuarioId: 2 },
]);

Atualizar Dados

Atualizar um registro existente.

SQLite Drizzle ORM 
UPDATE posts SET atualizado_em = '2025-03-28 00:00:00' WHERE id = 1;
 
await db
  .update(posts)
  .set({ atualizadoEm: '2025-03-28 00:00:00' })
  .where(eq(posts.id, 1));

Excluir Dados

Excluir um registro de uma tabela.

SQLite Drizzle ORM 
DELETE FROM usuarios WHERE nome_usuario = 'maria_silva';
 
await db.delete(usuarios).where(eq(usuarios.nomeUsuario, 'maria_silva'));

Linguagem de Consulta de Dados (DQL)

Selecionar Dados

Recuperar todos os registros de uma tabela.

SQLite Drizzle ORM 
SELECT * FROM usuarios;
 
const todosUsuarios = await db.select().from(usuarios);

Selecionar Colunas Específicas

Recuperar colunas específicas de uma tabela.

SQLite Drizzle ORM 
SELECT nome_usuario, email FROM usuarios;
 
const nomesEmails = await db.select({ nomeUsuario: usuarios.nomeUsuario, email: usuarios.email }).from(usuarios);

Filtro com WHERE

Filtrar resultados com base em condições.

SQLite Drizzle ORM 
SELECT * FROM usuarios WHERE nome_usuario = 'maria_silva';
 
const usuario = await db.select().from(usuarios).where(eq(usuarios.nomeUsuario, 'maria_silva'));

Ordenação (ORDER BY)

Ordenar os resultados por uma ou mais colunas.

Ordenar por nome_usuario em ordem ascendente (padrão).

SQLite Drizzle ORM 
SELECT * FROM usuarios ORDER BY nome_usuario;
 
const usuariosOrdenados = await db.select().from(usuarios).orderBy(usuarios.nome_usuario);

Ordenar por data_nascimento em ordem descendente.

SQLite Drizzle ORM 
SELECT * FROM usuarios ORDER BY data_nascimento DESC;
 
const usuariosOrdenados = await db.select().from(usuarios).orderBy(desc(usuarios.data_nascimento));

Ordenar primeiro por data_nascimento em ordem descendente e, em seguida, por email em ordem ascendente.

SQLite Drizzle ORM 
SELECT * FROM usuarios ORDER BY data_nascimento DESC, email ASC;
 
const sortedByDateAndEmail = await db.select().from(usuarios) .orderBy(desc(usuarios.data_nascimento), usuarios.email);

Paginação (LIMIT e OFFSET)

Recuperar os primeiros 10 usuários.

SQLite Drizzle ORM 
SELECT * FROM usuarios LIMIT 10;
 
const usuarios = await db.select().from(usuarios).limit(10);

Pular os primeiros 10 usuários e recuperar os próximos 10.

SQLite Drizzle ORM 
SELECT * FROM usuarios LIMIT 10 OFFSET 10;
 
const pagina2 = await db.select().from(usuarios).limit(10).offset(10);

Ordenação e Paginação Juntas

Ordenar usuários pelo nome_usuario e paginar os resultados para obter os primeiros 10 registros.

SQLite Drizzle ORM 
SELECT * FROM usuarios ORDER BY nome_usuario ASC LIMIT 10 OFFSET 0;
 
const primeirosUsuarios = await db.select().from(usuarios).orderBy(asc(usuarios.nomeUsuario)).limit(10).offset(0);

Paginação por páginas:

SQLite Drizzle ORM 
-- Página 1
SELECT * FROM usuarios ORDER BY nome_usuario ASC LIMIT 10 OFFSET 0;

-- Página 2
SELECT * FROM usuarios ORDER BY nome_usuario ASC LIMIT 10 OFFSET 10;

-- Página 3
SELECT * FROM usuarios ORDER BY nome_usuario ASC LIMIT 10 OFFSET 20;
 
// Página 1
const pagina1 = await db.select().from(usuarios).orderBy(asc(usuarios.nomeUsuario)).limit(10).offset(0);

// Página 2
const pagina2 = await db.select().from(usuarios).orderBy(asc(usuarios.nomeUsuario)).limit(10).offset(10);

// Página 3
const pagina3 = await db.select().from(usuarios).orderBy(asc(usuarios.nomeUsuario)).limit(10).offset(20);

Joins (Junções de Tabelas)

INNER JOIN

Combinar registros de duas tabelas com base em uma condição.

SQLite Drizzle ORM 
SELECT posts.titulo, usuarios.nome_usuario
FROM usuarios
INNER JOIN posts ON usuarios.id = posts.usuario_id;
 
const postsComUsuarios = await db
  .select({ titulo: posts.titulo, nome_usuario: usuarios.nome_usuario })
  .from(posts)
  .innerJoin(usuarios, eq(posts.usuario_id, usuarios.id));

LEFT JOIN

Retornar todos os registros da tabela à esquerda e os correspondentes da direita.

SQLite Drizzle ORM 
SELECT posts.titulo, usuarios.nome_usuario
FROM usuarios
LEFT JOIN posts ON usuarios.id = posts.usuario_id;
 
const postsComUsuarios = await db
  .select({ titulo: posts.titulo, nome_usuario: usuarios.nome_usuario })
  .from(usuarios)
  .leftJoin(posts, eq(usuarios.id, posts.usuario_id));

RIGHT JOIN (Simulado)

O SQLite não suporta RIGHT JOIN, mas é possível inverter a ordem das tabelas usando LEFT JOIN.

SQLite Drizzle ORM 
SELECT posts.titulo, usuarios.nome_usuario
FROM posts
LEFT JOIN usuarios ON usuarios.id = posts.usuario_id;
 
const postsComUsuarios = await db
  .select({ tituloPost: posts.titulo, nomeUsuario: usuarios.nomeUsuario, })
  .from(posts)
  .leftJoin(usuarios, eq(posts.usuarioId, usuarios.id));

Agregação e Agrupamento

Contagem

Conta o número de linhas.

SQLite Drizzle ORM 
SELECT COUNT(*) FROM usuarios;
 
const resultado = await db.select({ count: count() }).from(usuarios);
const totalUsuarios = resultado[0].count;

Agrupar por

Agrupa linhas com base em uma ou mais colunas.

SQLite Drizzle ORM 
SELECT usuario_id, COUNT(*) FROM posts GROUP BY usuario_id;
 
const postsPorUsuario = await db .select({ usuarioId: posts.usuarioId, count: count(), }) .from(posts) .groupBy(posts.usuarioId);

Soma

Soma os valores de uma coluna.

SQLite Drizzle ORM 
SELECT SUM(total) FROM pedidos;
 
const somaPedidos = await db .select({ totalSoma: sum(pedidos.total), }) .from(pedidos);

Máximo/Mínimo

Encontra o maior ou menor valor em uma coluna.

SQLite Drizzle ORM 
SELECT MAX(data_nascimento) FROM usuarios;
SELECT MIN(data_nascimento) FROM usuarios;
 
const dataMaxima = await db .select({ maxData: max(usuarios.dataNascimento), }) .from(usuarios);
const dataMinima = await db .select({ minData: min(usuarios.dataNascimento), }) .from(usuarios);

Transações

pode mudar account por usuarios e balance por pontos

await db.transaction(async (tx) => {
  const [account] = await tx.select({ balance: accounts.balance }).from(accounts).where(eq(users.name, 'Dan'));
  if (account.balance < 100) {
    // This throws an exception that rollbacks the transaction.
    tx.rollback()
  }
  await tx.update(accounts).set({ balance: sql`${accounts.balance} - 100.00` }).where(eq(users.name, 'Dan'));
  await tx.update(accounts).set({ balance: sql`${accounts.balance} + 100.00` }).where(eq(users.name, 'Andrew'));
});

SQLite Drizzle ORM
